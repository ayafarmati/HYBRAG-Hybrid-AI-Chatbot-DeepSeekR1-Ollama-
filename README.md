# HYBRAG-Hybrid-AI-Chatbot-DeepSeekR1-Ollama-
HYBRAG est un chatbot intelligent basé sur une approche hybride, combinant un modèle de langage (DeepSeek R1) et un système de recherche documentaire (RAG). Il s’appuie sur Ollama pour la génération d’embeddings et une base vectorielle afin de fournir des réponses contextualisées, fiables et basées sur des documents réels.
